# Ícones PWA

Adicione aqui os ícones do app:

- `icon-192.png` — 192x192px (obrigatório)
- `icon-512.png` — 512x512px (obrigatório, usado como maskable)

## Como gerar

1. Crie uma imagem quadrada de alta resolução (mínimo 512x512px)
2. Acesse https://realfavicongenerator.net ou https://maskable.app
3. Faça o upload e baixe os tamanhos necessários
4. Coloque os arquivos aqui nessa pasta

## Maskable icon

O icon-512.png deve ter a logo dentro de uma "safe zone" de 80% do centro
para funcionar corretamente em todos os launchers Android.
Use https://maskable.app/editor para verificar.
